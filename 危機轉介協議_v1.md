# 危機轉介協議 v1.0

> 鑑源心理研究室 | 2026-04-04
> 目的：當客戶展現心理危機信號時，確保鑑源團隊能安全、專業地回應並轉介

---

## 一、核心原則

**鑑源是命理陪伴平台，不是心理治療機構。**

- 我們可以看見、接住、陪伴
- 我們不可以診斷、治療、承諾解決心理問題
- 發現危機信號時，第一優先是**引導客戶到專業資源**，而不是試圖自己處理

---

## 二、觸發情境——高壓命格組合

以下命格組合出現時，報告撰寫者需特別注意語氣調整：

### 紅色警戒（必須調整語氣 + 附上轉介資源）

| 命格組合 | 潛在風險 | 語氣調整方向 |
|:---|:---|:---|
| 七殺+羊刃+流年沖克 | 衝動、暴力傾向、極端行為 | 強調「力量的正向出口」，避免提及衝突 |
| 太陰化忌+天機化忌 | 深度憂鬱、過度內耗 | 加重陪伴感，強調「你不是一個人」 |
| 命宮空宮+大運走空 | 存在感喪失、迷失方向 | 強調「留白是為了填入更好的東西」 |
| 孤辰+寡宿+華蓋+流年天哭 | 極度孤獨感、與世界斷開 | 強調「獨處是天賦」但加入連結建議 |
| 破軍化忌+流年太歲沖 | 生活全面崩解感 | 強調「破壞是重建的前奏」，給予具體穩定行動 |
| 八字梟印奪食 | 才華被壓制、表達困難、悶在心裡 | 強調「你的聲音值得被聽見」 |

### 黃色注意（需加強正向語氣）

| 命格組合 | 潛在風險 | 語氣調整方向 |
|:---|:---|:---|
| 大運走偏印+流年傷官 | 情緒波動大、易有極端想法 | 增加「這是暫時的」的時間框架 |
| 紫微獨坐+天空地空 | 高處不勝寒的孤獨 | 強調「高度帶來視野」 |
| 流年太歲刑沖+本命刑沖多 | 多重壓力疊加 | 具體列出「可以先做的一件事」 |
| 天同化忌+巨門化忌 | 失去快樂的能力、自我懷疑 | 強調小確幸和日常的美好 |

### 語氣調整通則

**遇到高壓命格組合時，報告中必須：**

1. **加重肯定語氣**：增加至少 2 處「你已經很好了」「你的存在本身就有意義」的語句
2. **降低壓力感**：不要列出太多「需要注意」的事項，聚焦在 1-2 個最重要的方向
3. **增加時間框架**：明確告訴客戶「這個階段會持續到什麼時候」，給予可預期的終點
4. **附上資源**：在報告結尾的「寫給你的話」中，自然地附上心理支持資源

## 三、報告語言調整指引——負面解讀轉為賦能敘述

### 轉化公式

```
傳統負面解讀 → 承認困難 → 命格釋罪 → 賦能重構 → 行動方向
```

### 實際範例

#### 範例 1：大運不利

| 步驟 | 內容 |
|:---|:---|
| 傳統解讀 | 「大運走食傷佩印，十年運勢低迷，事業不順，小人纏身。」 |
| 承認困難 | 「這十年確實不是一段輕鬆的旅程。你可能會感到怎麼努力都看不到回報，那種無力感是真實的。」 |
| 命格釋罪 | 「但你的命盤告訴我，這不是你的錯——你的大運正在走一段『深耕期』，就像種子在土壤裡的時間，看起來什麼都沒發生，但根系正在悄悄生長。」 |
| 賦能重構 | 「走過這段的人，往往在下一個大運會爆發出驚人的力量。你現在積累的每一份經驗和韌性，都是未來的燃料。」 |
| 行動方向 | 「在這段時間，最重要的事情不是追求外在的成就，而是：照顧好自己的身心、培養一項可以讓你心流的技能、維護好最核心的3-5段關係。」 |

#### 範例 2：感情不利

| 步驟 | 內容 |
|:---|:---|
| 傳統解讀 | 「命帶桃花劫，感情路多波折，晚婚為宜。」 |
| 承認困難 | 「感情一直是你心裡最柔軟也最容易受傷的地方。你渴望被真正理解，但總是在最投入的時候受傷最深。」 |
| 命格釋罪 | 「你的命盤顯示，不是你不會愛——而是你的靈魂需要更深層次的連結，淺薄的關係無法滿足你。那些走掉的人，不一定是你的錯，而是他們跟不上你的深度。」 |
| 賦能重構 | 「你命盤中感情的曲折，其實是在幫你篩選——篩掉不對的人，等到真正配得上你深度的那個人。」 |
| 行動方向 | 「現階段最重要的是：先成為讓自己感到安全的人。不是等別人來填滿你，而是先建立一份穩定的自我關係。」 |

#### 範例 3：健康警訊

| 步驟 | 內容 |
|:---|:---|
| 傳統解讀 | 「流年逢太歲，注意身體，恐有血光。」 |
| 承認困難 | 「今年你的身體可能會發出一些訊號——也許是容易疲倦、睡眠品質下降，或者免疫力不如從前。」 |
| 命格釋罪 | 「這不是什麼可怕的『命中注定』，而是你的身體在提醒你：你已經太久沒有好好照顧自己了。」 |
| 賦能重構 | 「從命盤來看，今年是你重新建立健康習慣的最佳時機。身體的訊號不是詛咒，而是邀請——邀請你回到與自己身體的連結。」 |
| 行動方向 | 「建議你做一件具體的事：預約一次完整的健康檢查。不是因為害怕，而是因為值得好好了解自己的身體。」 |

## 四、危機信號詞——客戶 Email / 訊息監測

### 紅色信號詞（立即啟動危機回應流程）

需要**立即回覆**（24小時內），附上專業資源：

```
一級關鍵詞（直接表達）：
- 不想活、想死、自殺、結束生命、活不下去
- 沒有活著的意義、活著好累、人間不值得
- 想消失、不想在這個世界了
- 傷害自己、自殘、割腕

二級關鍵詞（間接表達）：
- 他們沒有我會更好
- 一切都結束就好了
- 我撐不下去了
- 再也沒有希望了
- 如果我不在了
- 最後一次問
```

### 黃色信號詞（加強關注，回覆時附上關懷語句）

需要在**48小時內回覆**，加入額外的關懷段落：

```
- 很痛苦、非常崩潰、崩潰了
- 失眠很久了、整晚都睡不著
- 跟所有人都斷了聯繫
- 覺得自己是多餘的
- 沒有人理解我、沒有人在乎
- 每天都在哭、控制不了情緒
- 恐慌、喘不過氣、心臟很快
- 吃不下、完全沒食慾
```

### 橙色信號詞（暗示性表達，需結合上下文判斷）

```
- 好累（結合「活著」「人生」等語境時升級為黃色）
- 無所謂了（結合「一切」「人生」等語境時升級為黃色）
- 算了（反覆出現時升級為黃色）
- 沒關係（語境為否定自己的感受時注意）
```

## 六、客服回覆模板

### 模板 A：紅色信號——客戶表達想死或自殺意念

> 主旨：我們收到你的訊息了
>
> [客戶名字]，
>
> 謝謝你願意告訴我們你的感受。能把這些說出來，本身就需要很大的勇氣。
>
> 我們想讓你知道：你的感受是真實的，你的痛苦是被看見的。你不需要假裝沒事。
>
> 但同時，我們也想誠實地告訴你——你現在的狀態，需要比命理報告更專業的陪伴。不是因為你有什麼問題，而是因為你值得被真正專業地照顧。
>
> 我們強烈建議你現在就撥打以下電話，那裡有經過專業訓練的人，24小時都在等著接聽：
>
> - 台灣：1925 安心專線（免費，24小時）
> - 香港：2389 2222 撒瑪利亞防止自殺會（24小時）
>
> 你願意試試看嗎？打一通電話，聊聊就好。
>
> 我們在這裡，不會離開。你的報告我們會用最大的心意去完成，但現在最重要的事情是——讓你自己被好好照顧。
>
> 鑑源團隊

### 模板 B：黃色信號——客戶表達強烈痛苦或崩潰

> 主旨：我們聽到你了

> - 香港：2389 2222 撒瑪利亞防止自殺會（24小時）
>
> 記得照顧好自己。你的存在，比你以為的更重要。
>
> 鑑源團隊

### 模板 C：客戶對報告內容產生強烈負面情緒

> 主旨：關於你的報告——我們想跟你聊聊
>
> [客戶名字]，
>
> 謝謝你回饋你的感受。我們很重視你讀完報告後的每一個反應。
>
> 如果報告中的某些內容讓你感到不安或焦慮，我們想先跟你說：命盤呈現的是一種可能性和趨勢，不是不可改變的宣判。每一個命格特質都有它的光明面和陰影面，而你永遠有選擇的力量。
>
> 如果你希望，我們可以針對讓你不安的部分做更詳細的解說。你也可以回覆這封信告訴我們你的疑慮，我們會一一回應。
>
> 你的感受很重要，我們在這裡。
>

## 七、內部處理流程

### 收到危機信號後的標準作業程序

```
1. 識別信號等級（紅/黃/橙）
2. 紅色：
   a. 24小時內回覆，使用模板A
   b. 記錄事件到客戶檔案，標記「需關注」
   c. 報告撰寫時啟用「高壓命格語氣調整」
   d. 報告結尾必須附上轉介資源
3. 黃色：
   a. 48小時內回覆，使用模板B
   b. 報告撰寫時加強正向語氣
   c. 報告結尾建議附上轉介資源
4. 橙色：
   a. 正常流程回覆，但加入額外的關懷語句
   b. 持續觀察後續訊息
```

### 客服人員注意事項

- **不要說「我理解你的感受」**——沒有經歷過相同痛苦的人說這句話會顯得空洞。改說「我聽到你了」「你的感受是真實的」。
- **不要試圖解決問題**——我們的角色是引導轉介，不是治療。
- **不要承諾「一切都會好的」**——我們不知道會不會好，但我們可以說「我們在這裡」。
- **不要忽視任何信號**——寧可過度反應，也不要漏接。
- **每一次危機回應都要記錄**——用於內部學習和流程優化。
