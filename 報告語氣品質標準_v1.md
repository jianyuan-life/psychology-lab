# 報告語氣品質標準 v1.0

> 鑑源心理研究室 | 2026-04-04
> 目的：為所有報告的心理陪伴語言品質提供可衡量的標準

---

## 一、七大觸動技術——好例子 vs 壞例子

### 1. 鏡像時刻（Mirror Moments）

**核心**：說出客戶「自己知道但從沒被說出口」的話。

| 類型 | 範例 |
|:---:|:---|
| ✅ 好 | 「你是那種在人群中笑得最大聲，卻在回家關上門之後才感到真正的疲憊的人。」 |
| ✅ 好 | 「你不是不願意信任別人，而是你曾經全心全意相信過，結果被辜負了，所以現在每一次靠近都讓你本能地想後退。」 |
| ❌ 壞 | 「你的個性比較內向，不太擅長社交。」——太籠統，沒有觸碰到內心的真實感受。 |
| ❌ 壞 | 「你是一個很好的人。」——空洞的正面評價，沒有鏡像效果。 |

### 2. 正常化 × 命格釋罪（Normalization）

**核心**：用命盤解釋「你這樣是有原因的，不是你的錯」。

| 類型 | 範例 |
|:---:|:---|
| ✅ 好 | 「你總覺得自己不夠好、不夠努力——但你的命盤顯示，你天生就帶著一份對自己極高的標準。這不是缺陷，是你靈魂深處對卓越的渴望。」 |
| ✅ 好 | 「你在關係裡總是付出比較多，不是因為你傻，而是你的命格裡有一顆太陽——照亮別人是你本能的方式，只是你也需要學會讓別人照亮你。」 |
| ❌ 壞 | 「你的八字比肩過多，所以你容易跟人起衝突。」——只有診斷沒有釋罪，讀起來像被判了刑。 |
| ❌ 壞 | 「這是命中注定的，沒辦法改變。」——剝奪了主體性，製造無力感。 |

### 3. 依附理論語言（Attachment Theory）

**核心**：用焦慮型/迴避型/安全型的語言解釋感情模式。

| 類型 | 範例 |
|:---:|:---|
| ✅ 好 | 「你在感情裡常常覺得不安全——對方稍微不回訊息，你就開始想是不是自己做錯了什麼。這不是你太敏感，而是你內心有一個很深的需求：想被確認『你很重要』。」 |
| ✅ 好 | 「你不是冷漠的人，你只是習慣了用距離來保護自己。當有人走得太近，你的本能反應是往後退——因為親密對你來說，曾經意味著受傷。」 |
| ❌ 壞 | 「你屬於焦慮依附型人格。」——使用專業術語貼標籤，讓客戶覺得自己有問題。 |
| ❌ 壞 | 「你的感情問題是因為原生家庭創傷。」——越界分析，且可能觸發更多痛苦。 |

### 4. 無條件正向關懷（Carl Rogers UPR）

**核心**：全接納語氣，禁止評判性語言。

| 類型 | 範例 |
|:---:|:---|
| ✅ 好 | 「不管你做了什麼選擇，那都是你在當時能做的最好的選擇。你已經盡力了。」 |
| ✅ 好 | 「你可能覺得自己不夠好，但從你的命盤來看，你一直在用自己的方式努力著——只是你太習慣忽略自己的付出了。」 |
| ❌ 壞 | 「你應該要更積極一點面對問題。」——帶有評判和指導語氣。 |
| ❌ 壞 | 「你的問題在於太固執了。」——直接否定，違反全接納原則。 |

### 5. 脆弱性勇氣（Brene Brown）

**核心**：說出藏在行為背後的恐懼或羞恥感。

| 類型 | 範例 |
|:---:|:---|
| ✅ 好 | 「你把自己武裝得很好，因為你害怕的不是失敗——你害怕的是，如果拼盡全力還是不夠好，那就真的無處可逃了。」 |
| ✅ 好 | 「你不是不想談戀愛，你是害怕再一次把心交出去之後，發現對方根本不在乎。那種羞恥感，比孤獨更讓人難以承受。」 |
| ❌ 壞 | 「你太逞強了，要學會示弱。」——命令式語氣，忽略了防禦背後的原因。 |
| ❌ 壞 | 「你缺乏安全感。」——冷冰冰的診斷，沒有溫度。 |

### 6. 生命回顧與意義重建（Life Review）

**核心**：讓過去的痛苦有了意義和連貫性。

| 類型 | 範例 |
|:---:|:---|
| ✅ 好 | 「回頭看，那些你覺得走錯的路，其實都把你帶到了現在這個位置。那段最辛苦的時光，鍛造了你今天面對困難時那份淡定——這不是偶然，而是你一步一步走出來的。」 |
| ✅ 好 | 「你二十多歲時經歷的那些動盪，在命盤上看，正好是你人生最重要的淬煉期。那些經歷沒有白費——它們讓你擁有了大多數人沒有的深度和同理心。」 |
| ❌ 壞 | 「吃苦是必要的，苦盡甘來。」——雞湯式安慰，缺乏個人化的意義連結。 |
| ❌ 壞 | 「過去的事就讓它過去吧。」——否認過去經歷的重要性。 |

### 7. 情緒前置原則（Emotion-First）

**核心**：先映射情緒 → 命格釋罪 → 觸碰脆弱 → 給予方向。

| 類型 | 範例 |
|:---:|:---|
| ✅ 好 | 「最近這段時間，你可能常常感到一種說不清楚的焦慮（情緒映射）。這不是你想太多，你的命盤正在走一段轉折的運勢，內心的不安其實是你在為下一個階段做準備（命格釋罪）。那種『我不知道前方是什麼』的恐懼是真實的（觸碰脆弱），而你需要做的，就是允許自己慢下來，給自己多一點時間（方向）。」 |
| ✅ 好 | 「你一定很累了吧（情緒映射）。你的命格裡有一股天生的責任感，讓你永遠把別人放在自己前面（命格釋罪）。但在照顧所有人的過程中，你忘了問自己：你還好嗎？（觸碰脆弱）現在，也許是時候讓自己也被照顧一次了（方向）。」 |
| ❌ 壞 | 「你的運勢今年不太好，建議注意健康。」——直接跳到結論，沒有情緒鋪墊。 |
| ❌ 壞 | 「你命盤顯示你今年會有壓力，但下半年會好轉。」——冷冰冰的預測，毫無陪伴感。 |

## 二、報告開場白模板（5種命格類型）

### 類型 A：強勢型命格（七殺、比劫旺、獅子座日月等）

> 「你是那種永遠不願意被人看到脆弱面的人。在外面，你是大家眼中的強者、靠山、解決問題的人——但你心裡知道，有些時候你也很累，只是找不到一個可以安全放下盔甲的地方。
>
> 你的命盤告訴我一件事：你的堅強不是裝出來的，那是你骨子裡的東西。但同時，你的命盤裡也藏著一顆柔軟的心——只是你太習慣保護它了。
>
> 這份報告，不是要教你怎麼變更強，而是想讓你知道：你已經夠強了。接下來，我們一起看看，那些力量可以怎麼用得更聰明。」

### 類型 B：敏感型命格（正印重、太陰星、雙魚/巨蟹主星等）

> 「你一定常常覺得自己感受太多了——別人一個眼神、一句話的語氣變化，你都能接收到。這種敏感有時候讓你很疲憊，甚至會想：為什麼我不能像別人一樣粗線條一點？
>
> 但你的命盤告訴我，這份敏感是你最珍貴的天賦。你能感受到別人感受不到的東西，你能看到表面之下的真相。這個世界需要像你這樣的人。
>
> 這份報告會幫你理解，你的敏感從何而來，以及怎麼讓它成為你的力量，而不是負擔。」

### 類型 C：事業型命格（正官旺、天梁星、摩羯/處女主星等）

> 「你對自己的要求一直很高——不只是工作上的表現，更是對『人生應該走到哪裡』有一個清晰的畫面。但最近，也許你開始質疑：我這麼努力，真的在靠近我想要的生活嗎？
>
> 你的命盤顯示，你天生就帶著一份使命感。你不是那種可以隨波逐流的人，你需要覺得自己在做有意義的事。這份特質讓你走得比大多數人遠，但也讓你在迷茫的時候特別煎熬。
>
> 讓我們一起來看看，你的命盤為你規劃了一條什麼樣的路——以及你現在站在路上的哪個位置。」

### 類型 D：感情型命格（正財/偏財旺、紅鸞/天喜、金星/月亮強等）

> 「感情對你來說，從來不是生活的配角——它是你生命中最重要的主題之一。你渴望深刻的連結，渴望被真正理解，渴望那種『對的人就在身邊』的安心感。
>
> 但也許正因為你太在乎，每一次的失望都讓你傷得更深。你的命盤告訴我，你不是不懂得愛——你是愛得太認真了。
>
> 這份報告想幫你看見的是：你值得被愛的方式，以及什麼樣的關係模式最能讓你感到安全和幸福。」

### 類型 E：漂泊型命格（天機/破軍旺、驛馬星動、射手/水瓶主星等）

> 「你是那種靈魂裡住著一陣風的人——安定對你來說，有時候反而是一種壓力。你需要新鮮感，需要可能性，需要覺得『前方還有未知的風景在等我』。
>
> 但在漂泊的路上，你可能也會在某些深夜問自己：我到底在尋找什麼？是不是我一直在跑，是因為還沒找到一個值得停下來的理由？
>
> 你的命盤告訴我，你的不安定不是缺陷——那是你的靈魂在尋找屬於自己的答案。這份報告會幫你看清楚，什麼時候該勇敢往前走，什麼時候可以安心停下來。」

## 三、禁用語句清單

### 絕對禁止的語氣與用詞

| 類別 | 禁用語句範例 | 原因 | 替代方式 |
|:---|:---|:---|:---|
| **算命腔** | 「你命中注定...」「你命帶孤獨」「此命不利婚姻」 | 製造宿命感和無力感 | 「你的命盤顯示...這意味著你在這方面有特別的功課」 |
| **恐嚇語氣** | 「今年有血光之災」「小心車禍」「運勢極差」 | 引發焦慮，違反正向關懷原則 | 「這段時間需要多留意身體的訊號，好好照顧自己」 |
| **說教語氣** | 「你應該...」「你必須...」「你不能再...」 | 權威式指導，削弱主體性 | 「你可以考慮...」「也許可以試試...」「一個可能的方向是...」 |
| **比較語氣** | 「別人都能做到，你也可以」「很多人比你還慘」 | 否定個人感受的獨特性 | 「你的處境確實不容易」「你的感受是完全合理的」 |
| **空洞鼓勵** | 「加油！」「一切都會好的！」「船到橋頭自然直」 | 缺乏實質陪伴，令人感覺被敷衍 | 具體指出命盤中的轉機或可行動的方向 |
| **標籤化** | 「你就是太固執」「你的問題是不夠自信」 | 貼標籤等於判刑 | 描述行為背後的心理動力和原因 |
| **越界診斷** | 「你有焦慮症」「你可能有憂鬱傾向」 | 非專業診斷，可能造成傷害 | 「如果這些感受持續困擾你，建議和專業心理師聊聊」 |
| **否定感受** | 「別想太多」「沒什麼好擔心的」「這沒什麼」 | 讓客戶覺得自己的感受不重要 | 「你的擔心是可以理解的」「這種感受是真實的」 |
| **宿命論** | 「這就是你的命」「認命吧」「改不了的」 | 剝奪希望和行動力 | 「命盤指出這個挑戰，同時也指出了你克服它的天賦」 |

### 需謹慎使用的語句

| 語句 | 風險 | 使用條件 |
|:---|:---|:---|
| 「你的命盤比較特殊」 | 可能引發焦慮 | 後面必須接正面詮釋 |
| 「這段時間比較辛苦」 | 可能加重負面情緒 | 必須接「但你有能力走過」或轉機說明 |
| 「你的感情路比較曲折」 | 可能令人絕望 | 必須接「曲折的路往往通往最深刻的愛」等意義重建 |

## 四、報告心理品質評分標準（0-10分）

### 評分維度（5個維度，每項0-2分）

#### 維度 1：情緒映射度（0-2分）
報告是否準確映射了客戶可能的情緒狀態？

| 分數 | 標準 |
|:---:|:---|
| 0 | 完全沒有提及情緒，純命理技術分析 |
| 1 | 偶爾提及情緒，但不夠精準或深入 |
| 2 | 全文自然地映射客戶的情緒，讓人覺得「被看見了」 |

#### 維度 2：正向重構度（0-2分）
負面命理特質是否被轉化為成長語言？

| 分數 | 標準 |
|:---:|:---|
| 0 | 直接呈現負面解讀，或使用恐嚇性語言 |
| 1 | 有嘗試正向轉化，但略顯生硬或公式化 |
| 2 | 所有負面特質都被自然地轉化為天賦、功課或成長契機 |

#### 維度 3：個人化程度（0-2分）
語言是否針對這個客戶量身定制，而非通用模板？

| 分數 | 標準 |
|:---:|:---|
| 0 | 通用模板感強，換一個名字也適用 |
| 1 | 有結合命盤特質，但缺乏場景化的描述 |
| 2 | 高度個人化，結合了命格特質、人生階段、可能的生活場景 |

#### 維度 4：行動指引度（0-2分）
是否提供了具體、可執行的方向或建議？

| 分數 | 標準 |
|:---:|:---|
| 0 | 只有分析沒有建議，或只說「注意一下」 |
| 1 | 有建議但太籠統（如「注意健康」） |
| 2 | 提供具體的「第一步」或場景化的行動方向 |

#### 維度 5：觸動力（0-2分）
客戶讀完是否會有「胸口一緊、眼眶微熱」的感覺？

| 分數 | 標準 |
|:---:|:---|
| 0 | 讀完毫無感覺，像在看技術文件 |
| 1 | 有一些溫暖或共鳴，但不足以打動人心 |
| 2 | 至少有1-2處讓人「被擊中」的瞬間，想截圖或反覆閱讀 |

### 評分等級

| 總分 | 等級 | 是否可交付 |
|:---:|:---:|:---:|
| 9-10 | 卓越 | ✅ 可交付，且可作為範本 |
| 7-8 | 良好 | ✅ 可交付 |
| 5-6 | 及格 | ⚠️ 建議修改後交付 |
| 3-4 | 不及格 | ❌ 必須重寫關鍵段落 |
| 0-2 | 嚴重不及格 | ❌ 全文重寫 |

**最低交付標準：總分 7 分以上，且沒有任何維度為 0 分。**

## 五、品質檢查流程

### 報告生成後自動檢查清單

```
□ 禁用語句掃描 — grep 全文，確認無任何禁用語句
□ 情緒前置檢查 — 每個章節的第一段是否先映射情緒
□ 正向重構檢查 — 負面特質是否都有對應的正向轉化
□ 鏡像時刻計數 — 全文至少 3 處「你是那種...的人」式的精準描述
□ 行動指引計數 — 全文至少 5 處具體可行的建議
□ 觸動金句計數 — 全文至少 2 處「截圖級」的金句
□ 評分自查 — 五維度評分達 7 分以上
```
